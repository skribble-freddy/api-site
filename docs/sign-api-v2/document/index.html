Document

Endpoints for Documents

📄️ Upload document

Upload using a **Base64 encoded** file as `content`.

📄️ List all accessible documents

This request returns an array of all accessible `Documents` objects.

📄️ Get document metadata

By using the document ID you can fetch a `Document` metadata.

📄️ Delete document

Deleting a `Document` is similar to deleting a `SignatureRequest`.

📄️ Get document content

Dwnload the document content that contains the PDF document

📄️ Get document page preview

When uploading a PDF document to Skribble, the system generates some preview
